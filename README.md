# Table Code Block for Obsidian  

Table Code Block in Obsidian:  

- Add Row  
- Add Column  
- Edit Cell  
- Remove Row  
- Remove Column
- Set Head Row  
- Insert To Left Top Right Bottom  
- Quick Create Table  
- Render  
  - Headings  
  - Links  
  - Checkbox  
  - HTML  
  - Bold  
  - Italic  
  - Bold and Italic  
  - Block Quotes  
  - Lists  
  - Horizontal Rules  
- Translation  

## Not Support  

- Image  
- Code Block  
- Escaping Backticks
